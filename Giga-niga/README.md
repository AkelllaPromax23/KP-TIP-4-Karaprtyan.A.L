# Список фильмов - React приложение

Приложение для управления списком фильмов с возможностью отмечать любимые.

## Функциональность

- Просмотр списка фильмов
- Отметка фильмов как "любимые"
- Фильтрация по любимым фильмам
- Добавление новых фильмов
