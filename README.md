React-приложение "Список фильмов"
Я разработал React-приложение для управления списком фильмов с возможностью отмечать любимые фильмы.

Основные возможности
Отображение списка фильмов с названием, годом выпуска и рейтингом

Возможность отмечать фильмы как "любимые"

Фильтрация списка для просмотра только любимых фильмов

Добавление новых фильмов в коллекцию

Особенности реализации
Использование функциональных компонентов React с хуками (useState)

Локальное состояние для управления списком фильмов и фильтрацией

Компонентный подход для разделения логики и интерфейса

Обработка пользовательских событий (клики, ввод данных)

Условный рендеринг для отображения состояния "любимого" фильма

Компоненты приложения
MovieList - основной компонент для отображения списка фильмов

MovieItem - компонент отдельного фильма с кнопкой "любимый"

AddMovieForm - форма для добавления новых фильмов

FilterControls - элементы управления для фильтрации

Используемые технологии
React

CSS для стилизации

Локальное состояние (без внешнего API)
